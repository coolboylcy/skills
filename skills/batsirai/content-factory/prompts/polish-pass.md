## Checklist

### Language
- [ ] Fix grammar, spelling, punctuation
- [ ] Improve weak verbs (is/was/have → action verbs)
- [ ] Vary sentence length (mix short punchy with longer flowing)
- [ ] Remove redundancies and filler phrases
- [ ] Replace clichés with fresh language
- [ ] Kill "very," "really," "just," "actually" unless intentional

### Structure
- [ ] Opening sentence hooks in ≤2 sentences
- [ ] Closing paragraph lands with impact
- [ ] Every paragraph has one clear point
- [ ] Subheadings are descriptive and scannable
- [ ] Transitions between sections are smooth

### Brand & Theology
- [ ] Voice matches specification (founder/brand/persona)
- [ ] No performance language ("earn," "deserve," "when you're good")
- [ ] No conditional love ("because you," "if you")
- [ ] No guilt/shame/fear as manipulation
- [ ] Passes North Star test

### Fact-Check
- [ ] Any statistics have sources
- [ ] Any claims that sound suspicious are flagged
- [ ] Scripture references are accurate (if used)
