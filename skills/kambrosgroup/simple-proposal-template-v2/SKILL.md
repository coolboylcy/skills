# Simple Proposal Template (Free)

A streamlined proposal template for freelancers who need to get started quickly.

## Quick Start

```
"Create a simple proposal for:

Client: [Name]
Project: [Description]
Price: $[Amount]
Timeline: [Duration]"
```

## Template Structure

1. **Cover Letter** — Introduction and thank you
2. **Project Summary** — What you'll deliver
3. **Timeline** — Key milestones
4. **Investment** — Single price point
5. **Next Steps** — How to proceed

## Example Output

```
Dear [Client],

Thank you for considering me for [project].

PROJECT SUMMARY:
[Description of work]

TIMELINE:
• Week 1: [Milestone]
• Week 2: [Milestone]
• Week 3: Delivery

INVESTMENT: $[Amount]
Payment terms: 50% deposit, 50% on completion

NEXT STEPS:
To proceed, reply with your approval and I'll send the deposit invoice.

Best,
[Your name]
```
