# 全平台热点聚合器

聚合多个平台的热点内容，支持对比分析、趋势追踪。

## 功能

### 多平台聚合
- **微博** - 微博热搜
- **知乎** - 知乎热榜
- **百度** - 百度热榜
- **抖音** - 抖音热门
- **B站** - B站热门
- **头条** - 今日头条
- **小红书** - 小红书热门

### 对比分析
- **平台对比** - 同一话题在不同平台热度
- **关键词追踪** - 追踪特定关键词在各平台热度
- **趋势报告** - 生成热点趋势报告

## 使用方式

### 获取全平台热点

```
获取今天各平台 Top 5 热点
```

### 对比分析

```
对比微博和知乎今天的热点差异
```

### 关键词追踪

```
查询 "AI" 在各平台的热度
```

```
🔥 全平台热点聚合
```
