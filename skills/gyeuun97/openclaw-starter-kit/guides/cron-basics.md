# 크론잡 & 자동화 기초

## 하트비트 vs 크론

| 구분 | 하트비트 | 크론잡 |
|------|---------|--------|
| 타이밍 | ~30분마다 (유동적) | 정확한 시간 |
| 설정 | HEARTBEAT.md 편집 | 에이전트에게 말하기 |
| 용도 | 여러 체크 묶어서 | 단일 작업 정확히 |
| 예시 | "이메일+캘린더 체크" | "9시에 일정 알려줘" |

## 크론잡 설정 방법

에이전트에게 자연어로 말하면 됩니다:

```
"매일 아침 9시에 오늘 일정 알려줘"
"매주 월요일 10시에 주간 리포트 만들어줘"
"30분 후에 회의 시작이라고 알려줘"
"매일 저녁 6시에 오늘 한 일 정리해줘"
```

## 크론잡 관리

```
"크론잡 목록 보여줘"          → 설정된 작업 확인
"그 크론잡 꺼줘"              → 비활성화
"그 크론잡 삭제해줘"          → 완전 삭제
```

## 추천 자동화 (초보자용)

1. **아침 브리핑**: 매일 9시 → 날씨 + 일정 + 뉴스
2. **리마인더**: 특정 시간에 알림
3. **주간 리포트**: 매주 월요일 → 지난주 요약
4. **학습 루프**: 매일 관심 분야 자료 수집
