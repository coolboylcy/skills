# 百度地图 jsapi-ui-kit

轻量级百度地图 UI 组件库，提供地点自动补全、地点检索和详情展示能力。

## 什么时候使用

- 希望快速集成标准化的地图 UI 组件，无需从零开发
- 需要在地图应用中添加搜索输入框自动补全功能
- 需要在地图应用中添加地点搜索功能
- 需要展示 POI 详情信息（名称、地址、电话等）
