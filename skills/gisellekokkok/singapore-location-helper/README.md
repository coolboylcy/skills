## 功能亮点

### 1. 精确到楼栋的推荐
不只是"Tanjong Pagar"，而是：
- Icon Village（12 Gopeng St）- 商住混合， captive audience
- Duo Residences（7 Fraser St）- 地标建筑，设计感强
- Sky Suites @ Anson - 较新，设施好

### 2. 真实通勤分析
- 实测步行距离（不是广告距离）
- Shelter覆盖率（新加坡多雨，很重要）
- 早高峰拥挤程度（东西线 vs 环线）

### 3. 避坑指南
- 噪音：MRT轨道、马路、空调外机
- 朝向：西晒严重（空调费高20-30%）
- 施工：URA官网查周边开发计划
